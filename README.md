# 车辆管理系统简介 java361

## 项目概述

车辆管理系统是一款基于Springboot+Vue的前后端分离的Web应用。项目遵循MVC三层架构，结合了Spring、SpringBoot、Mybatis及Vue等现代技术开发，使用Maven作为项目管理工具，确保了系统的高效稳定。

## 技术栈

- 前端：Vue
- 后端：Springboot
- 数据库：MySQL
- 项目管理：Maven

## 功能模块

- 车辆信息管理
- 用户权限控制
- 数据统计与分析
- 系统日志记录

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- 前端：现代浏览器
- 后端：Java 8及以上版本，兼容Tomcat 8及以上服务器
- 数据库：MySQL 5.7及以上版本

## 部署步骤

1. 环境准备：确保Java和MySQL环境正确安装
2. 数据库部署：执行MySQL脚本，创建数据库及表结构
3. 后端部署：编译打包后部署到Java Web服务器
4. 前端部署：编译打包Vue项目，部署至Web服务器相应目录

## 目录结构

```
- backend                后端代码目录
  - src                  源代码目录
  - pom.xml              Maven项目文件
- frontend               前端代码目录
  - src                  源代码目录
  - package.json         Node.js项目文件
- database               数据库脚本及配置文件目录
```

## 核心亮点

- 前后端分离，便于集群和扩展
- 采用Mybatis进行数据持久化，保证了数据操作的高效性
- 完善的权限控制，确保系统安全
